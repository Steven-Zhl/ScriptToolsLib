## [电池状态报告.bat](./电池状态报告.bat)

> 使用cmd命令生成电池状态报告文件，并自动打开

### 环境要求

* Windows
* cmd

### 用法

>(3种，任选一种即可)

1. 双击直接运行
2. cmd中，直接输入`bat`文件的路径
3. Powershell中，输入`cmd /c [bat文件路径]`
