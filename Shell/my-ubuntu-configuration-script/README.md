# My-Ubuntu-Script

> 由于先前因为各种问题不止一次地需要重装Linux系统，所以自己写了个脚本，用于方便自己的系统配置，同样作为Shell的一个练手脚本

## 支持的发行版

* 最新版本：Ubuntu 23.04 (2023新版)
* 兼容版本：
  * Ubuntu 20.04

## 使用前提

* 已配置好网络环境(至少保证能够连接到国内的镜像源)
* x86_64架构(Arm架构的软件并没有测试过)

## 支持的软件

### 娱乐工具

* [x] Vlc

### 开发工具

* [x] Visual Studio Code
* [x] PyCharm (Community Version)
